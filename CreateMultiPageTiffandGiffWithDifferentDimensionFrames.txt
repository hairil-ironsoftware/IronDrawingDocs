# Create Multipage TIFF and GIF 
Users are able to combine AnyBitmap files into a multipage TIFF or GIF files in IronDrawing. Users can also use different dimensions of frame to create the multipage GIF and TIFF files.

To use this feature, all AnyBitmap frames need to be loaded using the file path and stored inside an initialized List of AnyBitmap type. The TIFF file can now be created from the frames stored by using <code>.CreateMultiFrameTiff(List<AnyBitmap>)</code> or use <code>.CreateMultiFrameGif(List<AnyBitmap>)</code> to create GIF file.
